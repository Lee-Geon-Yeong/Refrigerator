## 제품 기능과 가격
1. 온도 14도까지 내려감
2. 스마트폰 블루투스로 R,G,B, White 색상 제어 가능, on/off 기능으로 내부 LED 제어
3. 냉장고 온도와 습도를 알 수 있는 LCD 부착

원가 : 60000원 판매가 : 70000원
